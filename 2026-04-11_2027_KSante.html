3 Things to Work On (Detailed)

1. 0 control wards: five games, one rule change needed
Five games. Zero control wards. The number is consistent enough that it's clearly a decision pattern, not an oversight. K'Sante's position in top lane puts them adjacent to Baron pit every time they rotate mid/late. The fix is mechanical and takes no game knowledge to execute: buy a control ward on every back starting minute 15. Place it in the Baron pit brush. 75 gold. Five seconds to place. This is the single highest-ROI habit change available.
2. Using All Out (R) as a cross-map engage tool
Four games of 27–37% KP across Ornn and K'Sante suggests the pattern is in how roaming decisions are made, not the champion. Both champions have global or semi-global engage tools (Ornn R: global; K'Sante R: long-range dash/engage). When Blue team starts a fight in mid or bot lane, the correct play is to use All Out (R) to engage from the current position and start walking toward the fight. The R cast does not require being in melee range – it's a medium-range targeted dash that initiates the All Out state. Cast it toward the fight, then walk in.
3. Item completion before the last back
Five consecutive games with a component purchased but never completed. The pattern persists across champion changes (Ornn → K'Sante) which means it's in the shopping behavior, not the build path. Fix: on the back immediately before Baron/inhibitor push, identify the cheapest completion path in inventory and buy it entirely. Do not split gold between the component completion and a new component. Complete one item, then start the next. At 37–39 minutes with 2,000+ gold on hand, this is always achievable.
